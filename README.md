# AI Alter Ego - Personal Assistant

This project is an AI-powered personal assistant designed to represent my professional background and answer questions based on my own data, built during Week 1 of the **Complete Agentic AI Engineering Course** by Ed Donner on Udemy.

## Overview

AI Alter Ego is a tool-driven conversational assistant powered by the `meta/llama3-8b-instruct` model running on NVIDIA NIM. It reads and understands my résumé and LinkedIn data to provide context-aware responses about my experience, skills, and projects.

## Features

- AI assistant powered by LLaMA 3 8B Instruct model
- Tool-calling system to:
  - Record user details
  - Log questions that the assistant cannot answer
- Contextual understanding using résumé and LinkedIn profile data
- Structured conversational interface built with Gradio
- Fully deployed and accessible online via Hugging Face Spaces

## Use Cases

- Ask about my professional experience and skills
- Explore projects I’ve worked on
- Learn about the tools, technologies, and workflows I use
- Get quick summaries of my technical profile

## Technologies Used

- **Model:** `meta/llama3-8b-instruct`
- **Platform:** NVIDIA NIM (Neural Interactivity Model)
- **Interface:** Gradio for conversational UI
- **Deployment:** Hugging Face Spaces

## Getting Started

To run this project locally, you’ll need:

1. Access to the `meta/llama3-8b-instruct` model or equivalent
2. NVIDIA NIM infrastructure or compatible environment
3. Python 3.8+ environment with dependencies installed (e.g., Gradio)
